Amazon Q to OpenAI API - 快速开始
============================================================

【步骤 1】启动 API 服务
------------------------------------------------------------
cd C:\Users\QJHWC\Desktop\2api\amazonq2api
py -m uvicorn app:app --host 0.0.0.0 --port 8000

访问：http://localhost:8000

【步骤 2】添加账号（三种方式任选）
------------------------------------------------------------

方式A：全自动注册（最简单）⭐
  双击：开始注册.bat
  或在网页：http://localhost:8000 点击"🚀 启动全自动注册"

方式B：批量注册
  双击：批量注册.bat

方式C：手动 URL 登录
  网页：http://localhost:8000
  → 点击"开始登录"
  → 浏览器完成登录
  → 点击"等待授权并创建账号"

【步骤 3】测试 Chat
------------------------------------------------------------
网页：http://localhost:8000
→ Chat 测试区域
→ 点击"发送请求"

【完成】
账号已添加，可以使用 OpenAI 兼容 API 了！

API 地址：http://localhost:8000/v1/chat/completions
